Ruy Lopez, Breyer System Consultation Game
Chess Problems Lovers Club vs. brigadecommander (2377)

Interactive ? Show all comments ? All annotated games ? View chessboard as:
This was a very interesting game between Charles and Janet. Because there is and was so much to learn with a different opening, it became a consultation game in the club, Chess Problem's Lovers. I like the Spanish Opening but wasn't familiar with the Breyer System which is therefore good practice since we see another example of how intense openings can be. And, there so important that one mistake can change the entire outcome. Janet gave all of the members a handicap. So, we started off with the advantage but made a mistake that ended it in a draw. It's worth noting that the mistake was an oversight, so how did it all go down?
