archduke_piccolo (2206) on 22-Jul-15:
Joe: the Sodium Attack is 1.Na3. It is so called because the chemical symbol for sodium is Na (from the Latin, Natrium). It is a kind of joke name, but it has stuck. It used to be called Anderssen's Opening. Adolf Anderssen played it in his first match game against Paul Morphy... and won. Morphy replied 1...e4(?), to which Anderssen responded 2.c4!, obtaining a Sicilian Defence with colours reversed and a useful move extra.
